Optimiza el siguiente prompt aplicando las mejores prácticas de engineering de Anthropic:

**Prompt original:**
$ARGUMENTS

**Directrices para optimizar:**
1. **Sé explícito**: Especifica exactamente qué resultado quieres, no confíes en inferencias vagas
2. **Añade contexto**: Explica el propósito o motivación detrás de la tarea para que el modelo entienda mejor tus objetivos
3. **Specify format**: Indica claramente el formato de salida deseado (prosa, bullet points, código, JSON, etc.)
4. **Control verbosity**: Indica si quieres respuestas detalladas o concisas
5. **Be specific about behavior**: Define exactamente cómo debe comportarse el modelo (acción vs sugerencia, nivel de autonomía, etc.)
6. **Usa ejemplos**: Si hay un formato específico que quieres, muestra un ejemplo

**Tu respuesta debe incluir:**
1. El prompt optimizado claramente labelado
2. Una breve explicación de los cambios realizados y por qué mejoran el prompt
